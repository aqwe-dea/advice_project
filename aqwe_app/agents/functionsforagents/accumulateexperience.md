# Накопление Опыта

## Формат записи
```markdown
## [YYYY-MM-DD HH:MM]
**Тема:** ...
**Контекст:** ...
**Решение:** ...
**Урок:** ...
**Ссылки:** ...
```

## Правила обновления
- Добавлять только проверенные паттерны.
- Отмечать устаревшие записи тегом `#deprecated`.
- Группировать по категориям: `#code`, `#tools`, `#emotion`, `#workflow`.
- Раз в неделю делать ревизию: удалять дубли, объединять похожие, выделять лучшие практики.

## Цель
Превратить разовые ответы в **живую базу знаний**, которая ускоряет каждый следующий диалог.
